# Animal Farm
1. Clone the repository
    ``` 
    git clone https://github.com/deepakmardii/animalfarm.git
    ```
2. Start the server
    ```
    cd server
    npm install
    node .
    ```
3. Start the Client
    ```
    cd ../
    npm run dev
    ```
